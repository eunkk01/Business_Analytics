# 24-2 3학년 2학기

## 1. 수업 목적

- 마켓파인더 창업경진대회 참가
- 데이터 시각화 및 분석을 통한 인사이트 도출

<br>

## 2. 프로젝트 주제

- 취준생 및 기업을 위한 맞춤형 구인구직 추천 플랫폼

<br>

## 3. 담당 역할

- 데이터 전처리
- 취업 희망 지역 대시보드 제작 및 시각화
- 통계 검정 진행
- 희망지역 분류모델 구축

<br>
