# 24-1 3학년 1학기

## 1. 수업 목적

- 신용카드업의 위협요인 파악 및 그에 대한 대응 전략 수립 프로젝트

<br>

## 2. 프로젝트 주제

- 고객 특성 및 신용카드 분석에 따른 맞춤형 신용카드 혜택 조합 구축

<br>

## 3. 담당 역할

- 고릴라차트 데이터 수집
- 고릴라차트 및 자체 설문조사 데이터 전처리
- 고릴라차트 데이터를 활용한 인사이트 도출 및 시각화
- 카드 요소에 따른 특성 파악을 위한 클러스터링
- 카드 요소에 따른 순위 예측 모델

<br>
