# 24-2 3학년 2학기

## 1. 수업 목적

- 소비자 분석의 개념 및 방법 이해
- 실제 소비자 분석 주제 정의 및 방법을 의사 결정 과정에 활용

<br>

## 2. 프로젝트 주제

- 스마트스토어 네이버 광고를 통한 성과 도출

<br>

## 3. 담당 역할

- 광고 전략 수립
- 초기 스토어 분석
- 광고 집행 결과 분석 및 시각화
- PPT 제작

<br>
